# sql-data-warehouse-project
# SQL Data Warehouse – Retail Sales Analytics

## Project Overview
This project demonstrates an end-to-end SQL Data Warehouse using the
Bronze–Silver–Gold architecture. Raw CRM and ERP data is ingested,
cleaned, transformed, and modeled into a star schema for analytics.

## Tech Stack
- MySQL
- Power BI
- GitHub

## Architecture
- Bronze Layer: Raw data ingestion
- Silver Layer: Data cleaning and standardization
- Gold Layer: Star schema (Fact & Dimensions)

## Data Model
- Fact Table: fact_sales
- Dimensions: dim_customers, dim_products

## Business Insights
- Total sales by month
- Sales by product category
- Sales by country
- Average Sales
